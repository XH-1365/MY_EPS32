# ⚡ ESP32 基础学习简介(这个是我个人学习的ESP32库内容)

ESP32 = **Wi-Fi 📶 + 蓝牙 🔵 + GPIO + 多外设**  
结构稳定、能力强，适合 IoT 与创客项目。
---
## 🛠️ 开发方式
- **Arduino** — 最简单  
- **ESP-IDF** — 官方框架  
---
## 📚 基础学习路径
1. 🔌 GPIO 基础  
2. ✨ PWM / 定时器  
3. 🔗 UART / I2C / SPI  
4. 📡 Wi-Fi  
5. 🌐 HTTP / MQTT  
6. 🔄 FreeRTOS
